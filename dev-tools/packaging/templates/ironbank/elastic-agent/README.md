# elastic-agent

**elastic-agent** is a single, unified way to add monitoring for logs, metrics, and other types of data to each host. A single agent makes it easier and faster to deploy monitoring across your infrastructure. The agentâ€™s single, unified configuration makes it easier to add integrations for new data sources.
