The `audit` module reports security-relevant information based on data captured from the operating system (OS) or services running on the OS.

== file Fields

The file metricset generates events when a file changes on disk.



[float]
=== audit.file.path

type: keyword

The path to the file.

[float]
=== audit.file.action

type: keyword

example: attributes_modified

Action describes the change to the file. The possible values are: attributes_modified, created, deleted, updated, moved_to, collision_within, unmounted, root_changed.
