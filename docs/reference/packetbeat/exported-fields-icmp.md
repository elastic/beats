# ICMP fields [exported-fields-icmp]

ICMP specific event fields.

**`icmp.version`**
:   The version of the ICMP protocol.


**`icmp.request.message`**
:   A human readable form of the request.

type: keyword


**`icmp.request.type`**
:   The request type.

type: long


**`icmp.request.code`**
:   The request code.

type: long


**`icmp.response.message`**
:   A human readable form of the response.

type: keyword


**`icmp.response.type`**
:   The response type.

type: long


**`icmp.response.code`**
:   The response code.

type: long
