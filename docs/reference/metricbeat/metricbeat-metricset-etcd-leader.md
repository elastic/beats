# Etcd leader metricset [metricbeat-metricset-etcd-leader]

This is the leader metricset of the module etcd. This metrics is being read from the Etcd V2 endpoint and wonâ€™t show any activity regarding Etcd V3.

This is a default metricset. If the host module is unconfigured, this metricset is enabled by default.
