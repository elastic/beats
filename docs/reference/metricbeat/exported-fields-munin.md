# Munin fields [exported-fields-munin]

Munin node metrics exporter

**`munin.metrics.*`**
:   Metrics exposed by a plugin of a munin node agent.

type: object


**`munin.plugin.name`**
:   Name of the plugin collecting these metrics.

type: keyword
