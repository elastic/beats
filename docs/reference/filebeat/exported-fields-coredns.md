# Coredns fields [exported-fields-coredns]

Module for handling logs produced by coredns


## coredns [_coredns]

coredns fields after normalization

**`coredns.query.size`**
:   size of the DNS query

type: integer

format: bytes


**`coredns.response.size`**
:   size of the DNS response

type: integer

format: bytes
