# Host fields [exported-fields-host-processor]

Info collected for the host machine.

**`host.containerized`**
:   If the host is a container.

type: boolean


**`host.os.build`**
:   OS build information.

type: keyword

example: 18D109


**`host.os.codename`**
:   OS codename, if any.

type: keyword

example: stretch
