[[metricbeat-module-kubelet]]
== kubelet Module

beta[]

This module fetches metrics from Kubernetes https://kubernetes.io/docs/admin/kubelet/[kubelet] agent.


[float]
=== Example Configuration

The kubelet module supports the standard configuration options that are described
in <<configuration-metricbeat>>. Here is an example configuration:

[source,yaml]
----
metricbeat.modules:
- module: kubelet
  metricsets: ["node","container","volume","pod","system"]
  enabled: false
  period: 10s
  hosts: ["localhost:10255"]

----
