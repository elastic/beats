[[new-features-8.19.10]]
=== New features

The 8.19.10 release adds the following new and notable features.




**Filebeat**

* Add client secret authentication method for Azure Event Hub and storage in Filebeat. https://github.com/elastic/beats/pull/48289[#48289] https://github.com/elastic/beats/pull/48322[#48322] https://github.com/elastic/beats/pull/48324[#48324] https://github.com/elastic/beats/pull/48325[#48325] https://github.com/elastic/beats/pull/47873[#47873] 
* Improving input error reporting to Elastic Agent, specially when pipeline configurations are incorrect. https://github.com/elastic/beats/pull/47905[#47905] https://github.com/elastic/beats/issues/45649[#45649]
* Azure-eventhub-v2-add-support-for-amqp-over-websocket-and-https-proxy. https://github.com/elastic/beats/pull/47956[#47956] https://github.com/elastic/beats/issues/47823[#47823]
