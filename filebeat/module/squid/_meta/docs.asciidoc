== squid module

The +{modulename}+ module parses access logs created by the
https://squid.org[Squid Proxy] server.

include::../include/gs-link.asciidoc[]

[float]
=== Compatibility

The +{modulename}+ module was tested with logs from versions 3.5.20.

include::../include/configuring-intro.asciidoc[]

The following example shows how to set paths in the +modules.d/{modulename}.yml+
file to override the default paths for Squid Proxy Server access logs:

["source","yaml",subs="attributes"]
-----
- module: squid
  access:
    enabled: true
    var.paths: ["/path/to/log/squid/access.log*"]
-----

To specify the same settings at the command line, you use:

["source","sh",subs="attributes"]
-----
-M "squid.access.var.paths=[/path/to/squid/access.log*]"
-----
