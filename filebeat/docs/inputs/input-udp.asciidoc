=== UDP input

++++
<titleabbrev>UDP</titleabbrev>
++++

experimental[]

Use the `udp` input to read events over UDP.

Example configuration:

["source","yaml",subs="attributes"]
----
{beatname_lc}.inputs:
- type: udp
  max_message_size: 10KiB
  host: "localhost:8080"
----
